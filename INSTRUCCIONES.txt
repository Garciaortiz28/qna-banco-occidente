════════════════════════════

PASO 3 — INSTALAR DEPENDENCIAS
════════════════════════════════
  uv sync --python 3.12

PASO 4 — CONFIGURAR SUPABASE
══════════════════════════════
  1. Ir a supabase.com → tu proyecto → SQL Editor
  2. Pegar el contenido de supabase_setup.sql
  3. Clic en Run

PASO 5 — REINDEXAR ChromaDB
═════════════════════════════
  python main.py --step chunks
  (tarda 20-40 minutos — obligatorio al cambiar embeddings)

PASO 6 — PROBAR LA API
════════════════════════
  uvicorn api:app --host 0.0.0.0 --port 8000 --reload
  Abrir: http://localhost:8000/docs

PASO 7 — PROBAR CHAINLIT
══════════════════════════
  chainlit run chainlit_app.py
  Abrir: http://localhost:8000

PASO 8 — VERIFICAR TODO
═════════════════════════
  python diagnostico.py
